MATLAB program is included in the code folder. 
The program generates the an optimum path from start point to end points avoiding obstacles.

Steps to run the program
Extract the files to a folder
1) Open MATLAB and open [main.m] file which is in the Code folder.
2) Make sure all the files are in the code directory.
3) Change the working directory to Code folder.
4) Run the main program.
5) Enter the x and y cordinate for start and goal points.
6) The random nodes will be generated and will give a path once the execution is completed(reached the goal point).
7) Details regarding the nodes and nodeinfo are available at the variable workspace - cset - nodes and nodesinfo.
6) Output is also included in Output folder - OutputPlot & (Nodes and Nodesinfo)
